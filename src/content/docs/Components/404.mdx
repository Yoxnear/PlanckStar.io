Adding a custom 404 page gives personality to your website and is a nice touch. 
To add it, create a 404.astro file directly in the pages folder, import your 
layout, and include the component inside it.

```jsx
<!-- src/pages/404.astro -->

---
// Import your layout
import Layout404 from "../layouts/Layout404.astro";

//Import the page
import Simple404 from "../components/Simple404.astro"
---
<Layout404>
  <Simple404 />
</Layout404>

```
