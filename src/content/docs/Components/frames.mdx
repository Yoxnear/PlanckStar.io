Frames add an extra touch to your design and help highlight containers and sections.

You can apply them directly to your container—just place them inside, and the frame will automatically
adapt to the container’s dimensions. Also, the container must have the relative
class for it to work properly.

```jsx
<!-- src/components/Projects.astro -->

---
// Import
import Frame from "../components/Frame.astro";
---
(...)
<div class="relative + Type your container dimensions here">
  (...)
  <-- You can add it anywhere you want, as long as it’s inside the container -->
  <Frame />
  (...)
</div>
(...)

```
